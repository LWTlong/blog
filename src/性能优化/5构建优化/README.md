构建优化没什么好说的，就是打包工具的各种配置，各种优化，无非就是几个点

* 压缩代码，js, css, html，减少代码体积
* 压缩图片，减少图片体积
* 分包，提取公共代码和第三方库代码
* Tree Shaking，去除无用代码

关于构建优化，直接看 webpack篇章就行了 [webpack配置和优化](https://www.anbc.cn/blog/webpack/1%E5%9F%BA%E6%9C%AC%E9%85%8D%E7%BD%AE.html)
